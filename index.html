🤖 How the Agent Works

The Imperial Trade Sentinel agent answers natural‑language questions by:

  1. Fetching live telemetry from 59 microservices (portfolio, lithium, trade volume, etc.) running on a Samsung phone via Termux.
  2. Sending the question + context to Microsoft Azure OpenAI (Foundry IQ) for grounded reasoning.
  3. Returning a cited, verifiable answer.
curl -X POST http://localhost:8123/ask \
  -H "Content-Type: application/json" \
  -d '{"question": "What is our portfolio valuation and progress to R500B?"}'

✅ Real response (from our Azure OpenAI deployment):

  "answer": "The portfolio value is 269,911,161,297.38 (53.98% progress to R500B).",
